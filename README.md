# О проекте

Проект написан на React JS на шаблоне Create React App

## Библиотеки

Дополнительные библиотеки, которые используются: Axios, Char.js, Redux

## Запуск проекта

Выполнить следующие команды в терминале: сперва **npm install**, затем **npm start**
